# Sami Backend teste

## How to run the application?
```shell
  npm i
  npm start
```
And go to /documentation to see a swagger doc

## What is the stack?
 - Node 12.18.0
 - MongoDB

## Main dependencies
 - restify
 - joi
 - mongoDriver
 - restify-swagger-doc

## Endpoints exposed 
 - GET: **v1/beneficiary**
 - PUT: **v1/beneficiary**
 - PATCH: **v1/beneficiary**
 - POST: **v1/beneficiary**
 - DELETE: **v1/beneficiary**
